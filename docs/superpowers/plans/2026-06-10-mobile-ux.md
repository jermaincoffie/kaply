# Mobile & UX Refresh — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Voeg een bottom navigation toe aan kapper- en klant-layouts op mobiel, en fix 3 kritieke layout bugs + kapper afspraken card layout.

**Architecture:** Layouts-first aanpak — kapper.blade.php en klant.blade.php krijgen elk een Alpine.js-aangedreven bottom nav bar. Desktop sidebar/header blijft ongewijzigd (lg: breakpoint). Daarna worden views gefixed.

**Tech Stack:** Laravel 11, Livewire 3, TailwindCSS, Alpine.js (al aanwezig via Livewire)
